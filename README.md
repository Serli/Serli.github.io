# Installer Jekyll

(Nécessite Ruby)  
  *gem install jekyll*

# Lancer le serveur

pour lancer le serveur en local sur le port 4000 :  
  *jekyll serve --config _config.yml,_config_dev.yml -w*

# Ajout de Catégorie

Ajout dans le fichier : *data/group.yml*  
Puis, Création du dossier et de l'index associé dans : *Summary*
